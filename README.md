# Rhea Patel Portfolio

A personal portfolio website built to showcase my projects, experience, skills, resume, and career interests across software development, AI tools, product analysis, data, and user-centered design.

This site is designed to feel clean, polished, and personal while giving recruiters a clear view of what I have built, what I am learning, and where I am headed professionally.

## Live Site

[View Portfolio](https://rhea21204.github.io/rhea-patel-portfolio/)

## Features

- Responsive multi-page portfolio
- Light and dark mode toggle
- Custom hamburger menu
- Animated typing headline
- Project filtering
- Polished project cards
- Resume PDF link
- Soft blush, ivory, and espresso visual theme
